HTML das Grundgerüst einer Website

HTML ist die Sprache, mit der man die Struktur einer Webseite beschreibt und erstellt. Damit wird festgelegt, was auf der Seite zu sehen ist, z.B. Überschriften, Kopfzeilen, Bilder oder Links. HTML sorgt für die Struktur einer Website.

HTML in Kombination mit CSS

HTML ist besonders wichtig, da es das Grundgerüst einer Website bildet. Neben HTML wird in der Programmierwelt auch CSS verwendet. CSS wird verwendet, damit die Website nicht nur Inhalte anzeigt, sondern auch gut aussieht. Das bedeutet, dass CSS für das Design der Website verantwortlich ist.
